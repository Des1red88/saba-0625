## 💾 SQL код

```sql
-- Создание таблицы пользователей
CREATE TABLE users (
    id INT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    age INT
);

-- Выборка пользователей старше 25 лет
SELECT id, name, age
FROM users
WHERE age > 25
ORDER BY age DESC;

```
